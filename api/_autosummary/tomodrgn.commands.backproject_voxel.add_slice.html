tomodrgn.commands.backproject_voxel.add_slice#

add_slice(vol_ht: tensor, counts: tensor, ff_coord: tensor, ff: tensor) → None[source]#

Add one 2-D Hartley-transformed projection image as central slice to 3-D Hartley-transformed volume, modified in-place

Parameters:

  • vol_ht – torch tensor of 3-D Hartley-transformed volume without count scaling (modified in-place)

  • counts – torch tensor tracking weighting to be applied to each 3-D spatial frequency of vol_ht (modified in-place)

  • ff_coord – 3-D lattice coordinates at which to add Hartley-transformed image data for one image, centered at 0

  • ff – Hartley-transformed image data for one image
